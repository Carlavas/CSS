Ejemplo de Margen, Padding, Borde, Ancho y Alto

Este es un ejemplo de contenido dentro de un marco.

CSS Margin: Es el espacio entre el borde del marco y otros elementos.
CSS Padding: Es el espacio entre el contenido y el borde del marco.
CSS Border: Es la línea que rodea el marco.
CSS Width and Height: Controlan el tamaño del marco.
