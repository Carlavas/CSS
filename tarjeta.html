Explicación de CSS Margin, Padding, Border, Width y Height

CSS Margin: El margen es el espacio exterior que separa un elemento de otros en la página. No afecta al contenido ni al tamaño del elemento, solo lo separa de otros.

CSS Padding: El padding es el espacio interior que separa el contenido del borde del elemento. Si el padding es grande, el contenido estará más alejado del borde.

CSS Border: El borde es la línea que rodea al contenido y al relleno del elemento. Puedes definir su grosor, estilo y color.

CSS Width and Height: El ancho (width) y la altura (height) controlan el tamaño del área visible del elemento, sin incluir el margen, el padding ni el borde.
