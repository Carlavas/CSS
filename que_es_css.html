Introducción a CSS

1. ¿Qué es CSS y para qué sirve?

CSS (Cascading Style Sheets) es un lenguaje de diseño que permite separar el contenido de un sitio web de su presentación visual. Facilita la modificación de estilos en múltiples páginas a la vez, mejorando la eficiencia en el desarrollo. Además, ayuda a crear diseños responsivos y accesibles.

2. Formas de aplicar CSS en HTML

CSS en línea (inline)

Se aplican estilos directamente en el atributo style de un elemento HTML. Ejemplo: <p style="color: blue;">Texto en azul.</p>

CSS en el encabezado (internal)

Se utilizan etiquetas <style> en la sección <head> del HTML para aplicar estilos a toda la página. Ejemplo: 

<style>
    p { color: green; }
</style>

CSS externo

Se vincula un archivo CSS externo con <link> en la cabecera del HTML, ideal para múltiples páginas. Ejemplo: 

<link rel="stylesheet" href="styles.css">
